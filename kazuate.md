0～100からの数当てを考える。

# かたっぱし
- 0から数えると
  - 答えが0なら1回で当たる
  - 51なら51回
  - 100なら101回、の101通り
- それぞれ、101分の1の確率で起きるので、1/101 + 2/101 + ... + 101/101の合計が期待値になる
- 1から101までの合計は、(1+101) * 50 + 51 = 5100+51 = 5151
- 5101/101 = 51
- よって、当たるまでの期待値は51回

# ヒントがあるときの最適解
- 半分の値でチャレンジ
- 0～100 : 50で大きいか小さいか
  - 当たれば1回でBingo
  - 小さければ、正解は0～49の50のうちの1つ
  - 大きければ、正解は51～100の50のうちの1つ
- 小さいほうを検討 0～49
  - (24か)25でチェック
  - 25なら2回目で正解
  - ちいさければ、0～24の25のどれか
  - 大きければ、26～49の24のどれか
- 3: 小さいほうを検討 0～24
  - 13
  - 0～12
- 4: 小さいほう 0～12
  - 6
  - 0～5
- 5: 小さいほう 0～5
  - 3
  - 0～2
- 6: 小さいほう 0～2
  - 1
  - 0 or 2
- 7: 答えがわかる
- 以上から、1～7の正解が、それぞれ1/7で発生する
- よって期待値は、1/7 + 2/7 + ... + 7/7となる
- 28 / 7 = 4
- よって、当たるまでの期待値は4回
